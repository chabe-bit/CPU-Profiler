# CPU Sampling Profiler - Win32

This is an on-going attempt on implementing a low-level CPU sampling profiler for Windows that is built directly on top of ETW, inspired by Very Sleepy, a Windows CPU Profiler.

The attempt here is for a to better understanding how profilers work under the hood, where I may one day write a proper one myself.

Due to the heavily obsure documentation by Microsoft, this would be nearly impossible for anyone new to set their foot into. However, we have amazing figures such as Casey Muratori and Per Vognsen who can aid us:
